# 💜 Espaço Thyana Beleza

Site institucional do **Espaço Thyana Beleza**, desenvolvido para apresentar os serviços, portfólio de clientes e canais de contato da profissional Thyana Fontinelly — especialista em saúde e beleza capilar.

🔗 **[Acesse o site ao vivo](https://espaco-thyana-beleza.vercel.app)**

---

## ✨ Seções

| Seção | Descrição |
|---|---|
| **Intro** | Vídeo de abertura em tela cheia |
| **Quem sou eu** | Apresentação profissional com animação ao scroll |
| **Nossas clientes** | Carrossel contínuo com fotos das clientes |
| **Feedbacks** | Depoimentos de clientes |
| **Contato** | Links para Instagram e WhatsApp |

## 🚀 Rodando localmente

```bash
# Clone o repositório
git clone https://github.com/FabricioTP/espacoThyanaBeleza.git
cd espacoThyanaBeleza

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

Acesse em: `http://localhost:5173`

---

## 📦 Build para produção

```bash
npm run build
```

Os arquivos otimizados serão gerados na pasta `dist/`.
